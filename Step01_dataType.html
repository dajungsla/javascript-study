javascript data type

프로그래밍을 한다는 것은 결국 어떤 정보(data)를 다루는 것이다.

변수를 만들 때는 미리 약속된 예약어 let을 사용한다.
let 다음에는 변수의 이름을 정한다. 그리고 한 줄의 끝에는 ;을 작성한다.
=의 우측에 있는 값이 =의 좌측에 들어간다(대입된다). ex) let num=10;


- num이라는 기억공간(변수)에 10이 기억됨 / 값을 필요한 시점에 바꿀 수도 있다.
- 연산을 한 그 위치는 연산의 결과값으로 바뀐다(대체된다).

- 변수를 지정해주는 예약어 let / 연산자는 +? / 후루루루 type과 value를 구분하세요.
- 변수의 값은 필요한 시점에 바꿀 수 있지만, 변수의 중복 선언은 안됩니다!
- global area'(전역 변수)'
- object type{}이 필요한 이유는 여러 개의 데이터를 하나의 묶음으로 관리하고자 할 때 사용
